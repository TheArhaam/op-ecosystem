# decodeSentMessage

Decodes the payload emitted from the `SentMessage` event.

## Returns

### origin

- **Type:** `bigint`

### destination

- **Type:** `bigint`

### messageNonce

- **Type:** `bigint`

### sender

- **Type:** `Address`

### target

- **Type:** `Address`

### message

- **Type:** `Hex`

## Parameters

### payload

- **Type:** `Hex`

The payload to be executed emitted from the CrossL2ToL2DomainMessenge
